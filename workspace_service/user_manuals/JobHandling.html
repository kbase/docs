Accessing outstanding jobs

Lastly, the workspace provides commands to manage jobs submitted by the user to run on KBase cluster resources.

  1. kbws-jobs: retrieves a list of jobs submitted by the user to the workspace

1. kbws-jobs

 


	kbws-jobs [-?ehs] [long options...]
		-s --status      Job status (queued,running,done)
		-e --showerror   Set as 1 to show any errors in execution
		-? -h --help     Print this usage information



	$ kbws-jobs
		...
