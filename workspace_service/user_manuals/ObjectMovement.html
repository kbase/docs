Copying and moving workspace objects

Now that you understand how to create, retrieve, and revert workspace object, you will want to learn how to copy and move workspace objects. The workspaces allow you to copy any object you have read access to, and you can move any object you have write access to. There are two commands to support object copying and moving:

  1. kbws-copy: copies an object from one workspace and ID to another
  2. kbws-move: moves an object from one workspace and ID to another
