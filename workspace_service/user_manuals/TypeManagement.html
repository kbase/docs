Managing object types in the workspace

All objects stored in a KBase workspace are typed (e.g. Genome, FBAModel), and a recognizable type must be provided for all objects loaded into a workspace. You can see the list of types currently accepted by the workspace, and commands are provided to enable the addition of new types as needed by users. There are three commands to support the management of workspace types:

  1. kbws-types: retrieves a list of all types currently recognized by the workspace
  2. kbws-addtype: adds a new recognizable type to the workspace
  3. kbws-removetype: removes a user-added type from the list of recognized types

1. kbws-types


	kbws-types [-?eh] [long options...]
		-e --showerror   Set as 1 to show any errors in execution
		-? -h --help     Print this usage information



	$ kbws-types
		...
