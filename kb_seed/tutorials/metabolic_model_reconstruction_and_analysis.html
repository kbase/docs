Reconstruction and Analysis of Metabolic Models in KBase

Purpose: This tutorial describes the series of command-line scripts that provide access to model analysis capabilities in KBase.

Required Prerequisite Activities: Download KBase: Installing the KBase Application DMG, OR Get Started with Iris: Navigating Iris

Introduction

KBase includes numerous tools that enable the reconstruction and analysis of genome-scale metabolic models. Thus far, the following reconstruction and analysis algorithms have been implemented in the KBase:
  1. genome_to_fbamodel: enables the reconstruction of a metabolic model from an annotated genome object (the generation of which is covered in another tutorial)
  2. gapfill_fbamodel: gapfills a metabolic model to force growth in a specified media condition
  3. runfba: runs flux balance analysis on a metabolic model and returns a predicted flux profile
  4. fbamodel_to_html: prints a metabolic model to html format to facilitate reading the model
  5. fbamodel_to_sbml: prints a metabolic model to sbml format to facilitate analysis in other tools

genome_to_fbamodel

Command syntax: genome_to_fbamodel < myGenome.annotated > myModel.ini

gapfill_fbamodel

Command syntax: gapfill_fbamodel < myModel.ini > myModel.gapfilled

runfba

Command syntax: runfba < myModel.gapfilled > mySolution.fba

fbamodel_to_html

Command syntax: fbamodel_to_html < myModel.gapfilled > myModel.html

fbamodel_to_sbml

Command syntax: fbamodel_to_sbml < myModel.gapfilled > myModel.sbml

Summary

Here we describe the metabolic modeling tools currently implemented as web services in the DOE KBase.
