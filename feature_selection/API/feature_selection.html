NAME

FizzyClient

DESCRIPTION

MODULE: Fizzy This service receives a list of strings containing the options containing the metagenome IDs that will build the data matrix for feature selection. The SOURCE should be specified according to the available options specified by the KBase matrix service. The Fizzy service returns a list of strings contains the feature names that were selected.

feature_selection 

  $features = $obj->feature_selection($ids, $source)
Parameter and return types 
$ids is an IDs
$source is a SOURCE
$features is a FEATURES
IDs is a reference to a list where each element is a string
SOURCE is a string
FEATURES is a reference to a list where each element is a string
Description

FEATURE_SELECTION: TYPE function -- Run a Chi2 feature selection method on the abundance data contained in the metagenome IDs. Take in a list of IDs and the source used to form the feature vectors. Return the list of feature that were selected by the algorithm.

TYPES

IDs

Description

* IDs: TYPE list<string> * -- list of strings contain the metagenome IDs that are * either on KBase or MGRAST.

Definition 
a reference to a list where each element is a string

SOURCE

Description

Definition 
a string
