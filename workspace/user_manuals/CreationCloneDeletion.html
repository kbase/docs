Creating, cloning, and deletion of workspaces

Now that you can list the available workspaces and view workspace contents, it's time to learn how to create and delete you own workspaces. There are three commands to support the creation, deletion, and cloning of workspaces:

  1. kbws-createws: creates a new workspace to be owned and managed by you
  2. kbws-clone: creates an exact clone of an existing workspace to enable tinkering
  3. kbws-deletews: deletes an existing workspace with all objects (use with caution!!!)

1. kbws-createws

 


	kb_createws   [-?ehv] [long options...]
		-e --showerror   Show any errors in execution
		-v --verbose     Print verbose messages
		-? -h --help     Print this usage information



	$ kbws-createws myNewWorkspace r
		Workspace created with name:
		myNewWorkspace

2. kbws-clone

 


	kbws-clone   [-?ehw] [long options...]
		-w --workspace     ID for current workspace
		-e --showerror     Show any errors in execution
		-? -h --help       Print this usage information



	$ kbws-clone testClone r
		Workspace cloned to new workspace named:
		testClone

3. kbws-deletews

 


	kbws-deletews  [-?eh] [long options...]
		-e --showerror   Set as 1 to show any errors in execution
		-? -h --help     Print this usage information

Let's see an example of this command in action. ....


	$ kbws-deletews myNewWorkspace
		Workspace deleted with ID:
		myNewWorkspace
