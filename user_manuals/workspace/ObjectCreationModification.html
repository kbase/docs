Retrieving, altering, and loading workspace objects

Now that we can create workspaces and manage their permissions, we want to start populating our workspace with objects. Workspaces provide a online location for users to store and share their data, models, and analysis results. Most KBase tools operate on objects in the KBase workspace. There are four commands to support object retreival, load and deletion:

  1. kbws-get: enables the retrieval of object data from the workspace
  2. kbws-getmeta: enables the retrieval of object metadata from the workspace
  3. kbws-exists: checks if an object exists in a workspace
  4. kbws-load: enables new objects to be loaded into a workspace
  5. kbws-delete: enables the deletion of workspace objects
