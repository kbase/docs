NAME

GenomeAnnotation

DESCRIPTION

API Access to the Genome Annotation Service.

Provides support for gene calling, functional annotation, re-annotation. Use to extract annotation in formation about an existing genome, or to create new annotations.

METHODS

region_of_dna

Description

A region of DNA is maintained as a tuple of four components: 

                the contig
                the beginning position (from 1)
                the strand
                the length

           We often speak of "a region".  By "location", we mean a sequence
           of regions from the same genome (perhaps from distinct contigs).
Definition 
a reference to a list containing 4 items:
0: a contig_id
1: (begin) an int
2: (strand) a string
3: (length) an int

location

Description

a "location" refers to a sequence of regions

Definition 
a reference to a list where each element is a region_of_dna

annotation

Definition 
a reference to a list containing 3 items:
0: (comment) a string
1: (annotator) a string
2: (annotation_time) an int

feature

Description

represents a feature on the genome location on the contig with a type, and if a protein has translation, any aliases associated current history of annoation in style of SEED
