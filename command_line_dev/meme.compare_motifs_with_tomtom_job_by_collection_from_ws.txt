NAME
compare_motifs_with_tomtom_job_by_collection_from_ws - compare two collections of sequence motifs by TOMTOM.


VERSION
1.0

SYNOPSIS
compare_motifs_with_tomtom_job_by_collection_from_ws [--url=http://140.221.85.173:7077/ --ws=<workspace ID> --query=<MemePSPMCollection reference> --target=<MemePSPMCollection reference> --matrix=<PSPM ID> --thresh=<threshold> --evalue --dist=<allr|ed|kullback|pearson|sandelin> --internal --min_overlap=<value> --user=<username> --pw=<password>] 

DESCRIPTION
INPUT:            This command requires the URL of the service, references of two PSPM collections and parameters.

OUTPUT:           The output of this command is a Job object ID.

PARAMETERS:
--url             The URL of the service, --url=http://140.221.85.173:7077/, required.

--ws              Name of workspace where run result will be stored, required.

--query           Workspace reference of the query PSPM collection, required.

--target           Workspace reference of the target PSPM collection, required.

--matrix           KBase ID of a MemePSPM from the query collection that will be used. If omitted, all motifs in the query collection will be used.

--thresh          Only report matches with significance values â‰¤ thresh parameter value. 

--evalue          Use the E-value of the match as the significance threshold.

--dist            Motif distance calculation algorithm (accepable values are "allr", "ed", "kullback", "pearson", "sandelin").

--internal        This parameter forces the shorter motif to be completely contained in the longer motif.

--min_overlap     Only report motif matches that overlap by min overlap positions or more.

--user            User name for access to workspace.

--pw              Password for access to workspace.

--help            Display help message to standard out and exit with error code zero; 
                  ignore all other command-line arguments.  
--version         Print version information. 

 
EXAMPLES 
compare_motifs_with_tomtom_job_by_collection_from_ws --url=http://140.221.85.173:7077/ --ws=AKtest --query="AKtest/kb|memepspmcollection.2" --target="AKtest/kb|memepspmcollection.2" --thresh=0.0000001 --evalue --dist=pearson --internal --min_overlap=12

This command will return a Job object ID.
