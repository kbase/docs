Usage: net_entity_to_datasets [--url=http://kbase.us/services/networks] < enity_id


General options
	--url=[http://kbase.us/services/networks/]		the url of the service
	--help		print help information
	--version		print version information

Examples: 
echo 'kb|g.3899.locus.10' | net_entity_to_datasets

net_entity_to_datasets --help	print out help

net_entity_to_datasets --version	print out version information

Report bugs to kbase-networks@lists.kbase.us
