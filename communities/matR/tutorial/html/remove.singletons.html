remove.singletons {matR}R Documentation

Scrub Low-Abundance Entries from Matrix

Description

Replaces NA and single counts (or entries below a threshold) with zero, and removes rows of zeros (or rows with sum below a threshold).

Usage

remove.singletons(x, lim.entry = 1, lim.row = 1, ...)

Arguments

x

matrix

lim.entry

entry threshold for replacement with zero (numeric)

lim.row

row sum threshold for row removal (numeric)

...

currently unused

Value

The scrubbed matrix.

Author(s)

Kevin P. Keegan, Daniel T. Braithwaite
