asFile {matR}R Documentation

Write Metagenome and Analysis Objects to File

Description

Exports data. Methods are implemented or in progress for: collection, matrix, biom, boxplot, pco, parcoord, heatmap, sigtest. Text-format output is the intention here, but where a method is yet undefined, save() is used as the default, which writes a binary object image.

Usage

asFile (x, file, ...)

Arguments

x

object to write (any)

file

name of file (character)

...

currently unused

Author(s)

Daniel T. Braithwaite

See Also
