dependencies {matR}R Documentation

Install Packages Needed for Full Functionality

Description

Attempts to install certain add-on packages, some optional, others essential

Usage

dependencies(prompt = TRUE)

Arguments

prompt

TRUE for interactive use; FALSE for use in scripts

Details

This routine attempts to install, from repositories chosen by the user, the packages listed in the Suggests: field of the matR DESCRIPTION file. This is recommended. Functionality in matR is modular wherever possible, meaning that during interactive sessions add-on packages are not loaded until strictly necessary. Consequently, it is possible to work without the full set of suggested packages. On the other hand, matR readily incorporates further add-on packages as needed for specific analyses.
