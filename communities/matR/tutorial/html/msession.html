msession {matR}R Documentation

Session Configuration Including Authentication

Description

Accesses control details for the current session, including authentication, API server, and debugging info.

Author(s)

Daniel T. Braithwaite

See Also

Visualization, asFile

Examples

## Not run: 
# set and show current authentication key
msession$setAuth()
msession$setAuth(file="mykey.txt")
msession$getAuth()

# show available and current servers, and set
msession$servers()
msession$server()
msession$server("http://api.metagenomics.anl.gov")

# show recent API calls for debugging
msession$urls()

## End(Not run)
