mGet {matR}R Documentation

Access API Resources by ID

Description

Retrieves API resources with some helpful structure and processing

Usage

	mGet(resource = "matrix", x, with = NULL, ..., parse = TRUE, enClass = FALSE, file = NULL)

Arguments

resource

name of resource for call (length-one character)

x

resource IDs for call (character)

with

parameters for call (named list)

...

additional parameters

parse

parse result as JSON (logical)

enClass

convert to a structured object where possible (logical)

file

file name for retrieved resource (character)

Details

mGet provides structured access to API resources in an R-friendly manner. Most functionality of the API documented at http://api.metagenomics.anl.gov is available through this interface.

Retrieved resources may be files or data, and may or may not be transmitted in JSON. API resources implemented by mGet require an identifier (x) appropriate to the kind of resource requested.

In some calls, x can contain multiple values. Various formats for x are understood. See Identifiers for more details.

Authentication is required to access to private data.

Value

Various, according to the call issued.

Author(s)

Daniel Braithwaite

See Also

callRaw, Collections, Authentication
