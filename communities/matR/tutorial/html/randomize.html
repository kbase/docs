randomize {matR}R Documentation

Apply Function to Matrix Permutations for Randomization Tests

Description

Applies given function repeatedly to random permutation of a given matrix, providing a general-purpose routine to facilitate randomization tests.

Usage

randomize(x, ntimes = 1, method = c("sample", "dataset", "complete"), seed = NULL, FUN = identity, ...)

Arguments

x

matrix

ntimes

how many permutations (numeric)

method

type of permutation (character)

seed

seed for random number generation (numeric)

FUN

should accept single matrix argument and return summary value (function)

...

currently unused

Value

The form of results depends on the return value of the supplied summarizing function. By default, the random permutation matrices themselves are returned as a list, but a function should be supplied in place of the identity.

Author(s)

Kevin P. Keegan, Daniel T. Braithwaite
