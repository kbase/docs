default-views {matR}R Documentation

Data Attributes of Metagenome Collections

Description

Data objects provided to explain possible views of a metagenome collection

Usage

default.views
view.params

Details

view.params shows parameters and allowed values for specifying views of a collection. default.views shows the named views included by default, when a collection is created with no views explicitly specified.

Examples

default.views
default.views$raw

view.params
view.params$level$organism
