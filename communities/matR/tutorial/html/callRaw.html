callRaw {matR}R Documentation

Issue Any API Call Without Checks

Description

Processes input minimally to allow any API call, returning minimally processed results

Usage

callRaw(call, file = NULL)

Arguments

call

properly-formatted API call (character)

file

name of file for received data (character)

Details

Facilitates free-form API calls. Guidance regarding API syntax is available from the API page, linked below. The functionality of callRaw is limited to: prefixing the API server name, appending the current authentication key (if set), and writing results to a file (if specified).

Note: neither passing multiple calls/filenames, nor receiving filestreams is yet worked out.

For retrieval of private data, an authentication key must be set.

Author(s)

Daniel T. Braithwaite

See Also

http://api.metagenomics.anl.gov, Authentication, Identifiers, mGet, Collections

Examples

callRaw ("project/92")

t <- tempfile()
callRaw ("library/mgl53826", file = t)
unlink (t)
