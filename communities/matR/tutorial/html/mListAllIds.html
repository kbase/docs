mListAllIds {matR}R Documentation

List All IDs for a Specified Resource Type

Description

Lists all valid identifiers of accessible resources of a specified type

Usage

mListAllIds(resource = c ("project", "sample", "library", "annotation", "metagenome"))

Arguments

resource

kind of resource (character)

Details

Note that an authentication is required to access private resources.

Value

A character vector.

Author(s)

Daniel T. Braithwaite

See Also

Identifiers,Authentication,mGet

Examples

mListAllIds("project")
mListAllIds("sample")
mListAllIds("metagenome")
