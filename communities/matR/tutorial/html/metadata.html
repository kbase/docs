metadata {matR}R Documentation

Access Metadata of Collection

Description

Accesses metadata of an existing collection, or of specified metagenomes

Usage

metadata(x, ...)

Arguments

x

collection or metagenome resource ID(s) (collection or character)

...

currently unused

Details

Metadata elements are accessed in various ways, by field name. See below for examples.

Author(s)

Daniel T. Braithwaite

References

http://blog.metagenomics.anl.gov/metadata-in-mg-rast/,
http://gensc.org/gc_wiki/index.php/MIxS

See Also

Collections, metadata-class, metadata-methods

Examples

# collections have metadata
mm <- metadata(Waters)
mm <- metadata(collection(waters))

# retrieve only metadata retrieved for a set of IDs
mm <- metadata(guts)

# single index (length one)
mm["0464"]

# single index (length one)
mm["body product"]

# two indices (each length one)
mm["latitude", "longitude"]

# same, but return value is formatted as data.frame
mm["latitude", "longitude", bygroup=TRUE]

# single index (length two)
mm[c("0464", "env package.data")]

# three indices (each length two)
mm[c("0464", "PI"), c("0464","seq "), c("0464","biome")]
