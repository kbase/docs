Process Resource Identifiers

Description

Formats IDs properly. Returns resource types of given IDs.

Usage

scrubIds(ids, resources = c ("project", "library", "sample", "metagenome"))
scrapeResources(ids)

Arguments

ids

string(s) specifying resource ID(s) (character)

resources

string(s) specifying resource type(s) (character)

Details

These functions provide consistency in handling IDs to identify metagenomes and related resources.

scrubIds accepts IDs in various formats and returns a character vector of the same IDs properly formatted. resources should be length one, specifying one resource type for all IDs, or equal in length to ids, specifying the type of each. scrapeResources returns the resource types corresponding to IDs given in proper format as a character vector.

Value

A character vector.

Note

IDs given as numeric rather than character usually work, but use caution with metagenome IDs. Since they usually include a decimal point, problems can arise because as numerics they are float rather than integer.

Author(s)

Daniel T. Braithwaite

See Also

http://api.metagenomics.anl.gov, mGet, Collections

Examples

x <- c ("4445567.3", "4445568.3")
y <- "4445569.3;4445570.3"
z <- "4445571.3 mgp101 4445572.3"
A <- scrubIds ( c (x,y,z))
B <- scrubIds ("mgp101 mgp102 mgp103 mgp104 mgp105")
C <- scrubIds ("101;102;103", res = "project")
D <- scrubIds (c (22817, 22818, 22819), res = "samp")

scrapeResources (A)
scrapeResources (B)
scrapeResources (C)
scrapeResources (D)
l <- "mgl54248 mgl54250 mgl54252 mgl54254"
scrapeResources (l)
scrapeResources (scrubIds (l))
