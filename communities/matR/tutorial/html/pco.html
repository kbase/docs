pco {matR}R Documentation

Principal Coordinates Analysis of Collection

Description

Performs PCoA analysis, returning the results and plotting in two or three dimensions. This function wraps function pco() from package ecodist.

Usage

pco(x, ...)

Arguments

x

collection

...

further arguments as indicated below

Author(s)

Kevin P. Keegan, Daniel T. Braithwaite

See Also

pco-methods, pco-class, render

Examples

# basic plot
pco (Coral)

# color and label points according to a grouping
groups (Waters) <- c (rep (1, 15), rep (2, 9))
pco (Waters, comp = c (1,2), main = "fresh vs. spring water samples", col = c (rep ("blue", 15), rep ("red", 9)))

# plot three components
# note parameter "color" here; "col" above
col <- factor (metadata (Hospital) ["ventilation_type"])
levels (col) <- c ("red", "blue", "orange")
col <- as.character (col)
pco (Hospital, comp = c (1,2,3), color = col, labels = "")
