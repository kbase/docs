NAME

get_po_sampleidlist

SYNOPSIS

get_po_sampleidlist [--url=http://kbase.us/services/plant_expression_service] < poIDs

DESCRIPTION

Retrieve expression sample Ids (GSM#s) for a given list of plant ontologies.

Documentation for underlying call

Retrieve the list of expression samples (GSM#s) that corresponds to one or more of the plant tissue/organ type of interest. For example, if the input list of PO ids is:PO:0009049 then the output looks like: PO:0009049 Arabidopsis thaliana GSM133720 PO:0009049 Arabidopsis thaliana GSM133722 PO:0009049 Arabidopsis thaliana GSM133747 PO:0009049 Arabidopsis thaliana GSM133748 PO:0009049 Populus trichocarpa GSM327656 PO:0009049 Populus trichocarpa GSM327657 PO:0009049 Populus trichocarpa GSM327658 PO:0009049 Populus trichocarpa GSM327659 PO:0009049 Populus trichocarpa GSM327660 PO:0009049 Populus trichocarpa GSM327661

OPTIONS

-u [http://kbase.us/services/plant_expression_service] --url=[http://kbase.us/services/plant_expression_service] hostname of the server
--species species=Ara for Arabidopsis and Pop for Poplar
--help prints help information
--version print version information

EXAMPLE

 echo 'PO:0009047' | get_po_sampleidlist
 get_po_sampleidlist --help
 get_po_sampleidlist --version

VERSION

0.1
