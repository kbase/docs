NAME

 get_experiments_by_sampleid

SYNOPSIS

 get_experiments_by_sampleid [--url=http://kbase.us/services/plant_expression_service] < sampleIDs

DESCRIPTION

 This function provides the expression values for given experiment sample IDs (GSM#s).

Documentation for underlying call

This function provides the expression values corresponding to each given experiment sample in the input list of sample ids (GSM#s).For each sample in the input list of samples, it extracts the expressed genes (kbase gene identifier) and their corresponding expression values. e.g.for samples:"GSM128731","GSM131686" Output looks like this: "","GSM128731","GSM131686" "kb|g.3899.locus.23281",4.19051374151256,4.81544532287956 "kb|g.3899.locus.23342",8.0461118314498,7.75470811901751 "kb|g.3899.locus.23319",8.95008412287335,8.0937520447078 "kb|g.3899.locus.23295",6.70090411599666,7.42720023331292 "kb|g.3899.locus.23378",8.79308438662897,8.23643301305399

OPTIONS

-u [http://kbase.us/services/plant_expression_service] --url=[http://kbase.us/services/plant_expression_service] hostname of the server
--help prints help information
--version print version information

EXAMPLE

 echo GSM128731 | get_experiments_by_sampleid
 get_experiments_by_sampleid --help
 get_experiments_by_sampleid --version

VERSION

0.1
