NAME

get_po_descriptions

SYNOPSIS

get_po_descriptions [--url=http://kbase.us/services/plant_expression_service] < poIDs

DESCRIPTION

Retrieve selected plant ontologies description

Documentation for underlying call

 Retrieve the list of selected plant ontology IDs (POIDs) descriptions corresponding to an input list of POIDs. For example if the input list=PO:0000003,PO:0000005,PO:0000006
 then the output looks like:
 PO:0000003     whole plant
 PO:0000005     cultured cell
 PO:0000006     cultured protoplast

OPTIONS

-u [http://kbase.us/services/plant_expression_service] --url=[http://kbase.us/services/plant_expression_service] hostname of the server
--help prints help information
--version print version information

EXAMPLE

 echo 'PO:0000006' | get_po_descriptions
 get_po_descriptions --help
 get_po_descriptions --version

VERSION

0.1
