getExperimentsByExperimentID

  $results = $obj->getExperimentsByExperimentID($ids)
Parameter and return types
$ids is an ExperimentIDList
$results is an Experiments
ExperimentIDList is a reference to a list where each element is an ExperimentID
ExperimentID is a string
Experiments is a reference to a hash where the key is an ExperimentID and the value is an Experiment
Experiment is a reference to a hash where the following keys are defined:
	series has a value which is a Series
	genes has a value which is a GeneIDList
Series is a reference to a hash where the key is a SeriesID and the value is a ValueList
SeriesID is a string
ValueList is a reference to a list where each element is a float
GeneIDList is a reference to a list where each element is a GeneID
GeneID is a string
Description

This function retrieves the experiment series (typically NCBI GEO series ids: GSE#s), expressed genes and their corresponding expression values for each given experiment in the input list of experiment ids (typically each item in the list is a GSM#). It then returns a table of data containing information such as GSE#, GSM#, Gene ID, Expression Value.
