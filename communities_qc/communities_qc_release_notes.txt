OVERVIEW
-----------------------------------------
This KBase service provides a server for asynchronous computation of quality information for metagenomic sequences. It produces kmer, drisee and histogram statistics.

Technical documentation is available in docs/communitiesQC.html. It can be re-generated by running 'make build-docs'. Deployment documentation is available in the README file.
