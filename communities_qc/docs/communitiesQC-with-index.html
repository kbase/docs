module Microbial Communities QC

The microbial communities Quality Control Server allows the computation of drisee profiles, k-mer profiles and histograms via an asynchronous API. There are methods for each statistic, instance and compute. The compute method will trigger a compute for a specified sequence. If a compute for that sequence is already running, it will return the status of that compute. Upon completion the results can be obtained through the instance method.

When using the client scripts, note that the parameters described here must be passed as a hash, not an ordered list, e.g. in perl:

my $drisee_result = get_drisee_instance( { id => $my_drisee_id } );

get_drisee_compute

Description

quality assessment of metagenomic sequence computes quality assessment data of a metagenomic sequence

Parameters

  • string id

    id of the metagenome to be analyzed

Return Attributes

  • string time

    computation time of the job at the time of the query

  • string status

    status of the compute, one of [ waiting, running, error, complete ]

  • string error

    the error message if one ocurred, otherwise an empty string

  • string id

    id of the metagenome to be analyzed

get_drisee_instance

Description

quality assessment of metagenomic sequence Returns a single object.

Parameters

  • string id

    unique object identifier

Return Attributes

  • string version

    Version of the drisee score

  • mapping<string, string> substitution

    hash ref

  • float insertion_delition

    insertion deletion quotient

  • string created

    Creation time for this drisee resource

  • string id

    Unique ID of the sequence file calculated the drisee error from

  • float total_errors

    cumulative drisee error
