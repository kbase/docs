NAME

Bio::KBase::interlog_projection::Client

DESCRIPTION

Interlog Projection Service APIs

This module provides services for getting interlog network of species A projected from the query network of species B based on ortholog information. The ortholog information is precomputed by Inparanoid algorithm and we plan to provide KBase Tree service based projection. It will serve queries for various types of network transfer from one species to the other species network. (i.e., co-expression, PPI, metabolic pathway, functional association network) User can exclude link pair that show low ortholog confidence using cut-off value.

project_network_with_custom_methods

  $results = $obj->project_network_with_custom_methods($source, $target_species, $projection_method, $cut_off)
Parameter and return types
$source is a Network
$target_species is a string
$projection_method is a string
$cut_off is a float
$results is a Network
Network is a reference to a hash where the key is a string and the value is a string
Description

Get the interlog projected network using two different projection_method at this time: 'any' or 'bbh' -- bidirectional best hits. This function will support additional projection_method in the near future. source is a source network to be projected. target_species is a genome id to project projection_method is either 'any' or 'bbh' for one to many projection or bidirectional best hits. cut_off is a float value to filter out uncertain orthologies. For inparanoid orthology informations, 1.0 is equal to bbh.
