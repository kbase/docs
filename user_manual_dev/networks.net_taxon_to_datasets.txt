Usage: net_taxon_to_datasets [--url=http://kbase.us/services/networks] < taxon_id


General options
	--url=[http://kbase.us/services/networks/]		the url of the service
	--help		print help information
	--version		print version information

Examples: 
echo 'kb|g.3899' | net_taxon_to_datasets

net_taxon_to_datasets --help	print out help

net_taxon_to_datasets --version	print out version information

Report bugs to kbase-networks@lists.kbase.us
