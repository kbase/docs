Usage: net_network_type_to_datasets [--url=http://kbase.us/services/networks] < network_type


General options
	--url=[http://kbase.us/services/networks/]		the url of the service
	--help		print help information
	--version		print version information

Examples: 
echo 'PROT_PROT_INTERACTION' | net_network_type_to_datasets

net_network_type_to_datasets --help	print out help

net_network_type_to_datasets --version	print out version information

Report bugs to kbase-networks@lists.kbase.us
