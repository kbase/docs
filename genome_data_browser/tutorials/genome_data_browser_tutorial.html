Using the KBase Genome Information Browser

Purpose: This tutorial demonstrates how to browse some basic information regarding genomes stored in KBase.

Required Prerequisite Activities: None

Suggested Prerequisite Activities: Getting Started with KBase

Related Tutorials: Model Builder, Model Viewer, GLAMM

Splash Page

On entering the Genome Information Browser from KBase Labs, you will be greeted with a splash page describing the browser and linking to this tutorial. Click on the "Get Started" button to start the info browser.

Navigation Bar

The navigation bar exists on all Genome Information Browser pages.

  • Clicking the KBase logo in the upper-left will always return you to the KBase homepage.
  • The "Genome Info" link on the left side will always take you to the Genome Information Browser main page.
  • The About link takes you to a brief page containing general information about this webpage and KBase Labs.
  • The Contact link takes you to a page with ways to contact the author and maintainer of the Genome Information Browser.

Main page

On loading the main page of the Genome Information Browser, you will be greeted with a table listing all genomes stored in KBase. This table lists the KBase ID, scientific name, and biological domain of each genome. Clicking the header for each of these columns will sort the table by that column.

The search area on the upper right of the table allows you to view a subset of genomes as you type. For example, typing "arabidop" will show all Arabidopsis thaliana genomes as well as Arabidopsis lyrata. See the image below.


Finally, clicking any of the scientific names of the species will take you to the information page about that genome.

Genome Information Page

The Genome Information Page contains a summary of information stored in KBase regarding a selected genome. For this tutorial, we will use the example of Shewanella oneidensis MR-1

As in the image below, every genome information page has 3 main components:

  1. A title bar with the scientific name and a "return to index" link. The link will take you back to the table of all genomes.
  2. A table of genome information on the left.
  3. A contig browser on the right.


Genome Information Table

The general information on each genome is listed in this table, as follows:

  • ID - the genome's KBase ID.
  • Scientific Name - the scientific name of the species for this genome.
  • Domain - the biological domain of this species (eukaryote, bacteria, archaea, virus, etc.).
  • Complete? - This is marked "Yes" if the genome is completely sequenced and annotated.
  • Genome Size - The number of nucleotide base pairs in the genome.
  • GC Content - The combined percentage of guanine and cytosine nucleotides in the genome.
  • Number of Contigs - The number of contiguous subsequences of which the genome is composed in KBase.
  • Number of Features - The number of genetic features in the genome (genes, non-coding RNAs, promoter sequences, operons, etc.).
  • Number of Protein Encoding Genes - The number of features in the genome which are annotated as protein encoding genes.
  • Number of RNA Features - The number of features in the genome which are annotated as RNA.
  • Source - The original source of the genome and its ID in that source (e.g. MOL (MicrobesOnline), SEED, iPlant, Ensembl).
  • Taxonomy - The detailed taxonomic classificaiton of the genome.

Contig Browser

The contig browser is a simple widget that allows you to interactively view the feature content of a genome. It is used as follows.

First, select a contig from the dropdown menu at the top of the browser. If there is only one contig, then only one element will be present in the menu. Once a contig is selected, the first features present along the first 10,000 bases of the contig will be displayed as red arrows. The direction of the arrow shows the read direction of the feature: features that point to the right are read in the same direction as the genome sequence as stored in KBase, while features that point to the left are read in the opposite direction along the complementary strand of the stored sequence. Features that overlap each other are displayed on separate rows, as in the figure above. Featured composed of multiple regions have their regions linked by a horizontal line.

Clicking on a feature arrow will change that arrow's color from red to yellow, and show some simple information regarding that feature. This includes:

  • Feature ID - the KBase feature ID. Clicking this ID will take you to the feature information page.
  • Function - the annotated function of the feature.
  • Length - the total number of base pairs in the feature.
